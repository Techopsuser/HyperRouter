# HyperRouter

**The Universal AI Router for Free & Open LLMs**

One API • Multiple Providers • Intelligent Routing • Automatic Fallback
• OpenAI Compatible

------------------------------------------------------------------------

## Overview

HyperRouter is a production-grade Universal AI Router that intelligently
routes requests across 70+ free and open Large Language Models through a
single OpenAI-compatible API.

### Key Features

-   Intelligent model routing
-   Automatic intent detection
-   Multi-provider support
-   Automatic fallback
-   Health monitoring
-   Benchmark engine
-   Streaming responses
-   OpenAI-compatible API
-   Analytics dashboard
-   Redis caching
-   PostgreSQL
-   Docker & Kubernetes ready
